
J'ai poussé le template sur GitHub en utilisant Git LFS, car sa taille dépasse la limite autorisée.

Pour le récupérer, suivez ces étapes :

1- Installer Git LFS

2- git clone "https://......."

3- git lfs install

4- git lfs pull


Vous pouvez également télécharger directement le fichier ".rar" et commencer à l’exploiter en local.
